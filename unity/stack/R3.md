# R3 Reactive Extensions

> **Scope**: R3 reactive programming — observables, subscriptions, disposal, ReactiveProperty, EveryUpdate.
> **Load when**: reactive streams, Observable, Subscribe, ReactiveProperty, CompositeDisposable, AddTo, R3.

---

## Subscriptions & Disposal

ALWAYS unsubscribe from Observable.

### MonoBehaviour

```csharp
// Single subscription — AddTo(this)
public class Timer : MonoBehaviour
{
    void Awake()
    {
        Observable.Timer(TimeSpan.FromSeconds(1))
            .Subscribe(x => Debug.Log("Hello World!")).AddTo(this);
    }
}

// Multiple subscriptions — AddTo(_disposables)
// Before re-subscribing ALWAYS unsubscribe via _disposables.Clear()
public class Timer : MonoBehaviour
{
    private readonly CompositeDisposable _disposables = new();

    public void Subscribe()
    {
        _disposables.Clear();

        Observable.Timer(TimeSpan.FromSeconds(1))
            .Subscribe(x => Debug.Log("Hello World!")).AddTo(_disposables);
    }
}
```

### Plain Classes

```csharp
// Unsubscribe via IDisposable and _disposables.Dispose()
public class Timer : IDisposable
{
    private readonly CompositeDisposable _disposables = new();

    public Timer()
    {
        Observable.Timer(TimeSpan.FromSeconds(1))
            .Subscribe(x => Debug.Log("Hello World!")).AddTo(_disposables);
    }

    public void Dispose()
    {
        _disposables.Dispose();
    }

    // For re-subscription: _disposables.Clear() before subscribing
    public void Subscribe()
    {
        _disposables.Clear();

        Observable.Timer(TimeSpan.FromSeconds(1))
            .Subscribe(x => Debug.Log("Hello World!")).AddTo(_disposables);
    }
}
```

## Observable.EveryUpdate

Replace MonoBehaviour Update with Observable.EveryUpdate:

```csharp
// Plain class with Observable.EveryUpdate(UnityFrameProvider.Update)
public sealed class CooldownManager : IDisposable
{
    private readonly CompositeDisposable _disposables = new();

    public CooldownManager()
    {
        Observable.EveryUpdate(UnityFrameProvider.Update)
            .Subscribe(_ => UpdateCooldowns())
            .AddTo(_disposables);
    }

    public void Dispose() => _disposables.Dispose();

    private void UpdateCooldowns() { /* ... */ }
}
```

## Reactive Properties

Always expose reactive properties via ReadOnly:

```csharp
public sealed class SimpleTimer : IDisposable
{
    private readonly CompositeDisposable _disposables = new();
    private readonly ReactiveProperty<float> _value = new();

    public ReadOnlyReactiveProperty<float> Value => _value;

    public void Dispose() => _disposables.Dispose();
}
```

If someone subscribes to reactive property values, there MUST always be mandatory unsubscription as described above.
