# Rules Catalog

A complete registry of rules shipped by this repository, grouped by game engine. Each table links directly to the rule file so you can open it and read the full body.

- Rule IDs and versions come from each rule's YAML frontmatter.
- Descriptions are the canonical short descriptions mirrored in [`manifest.json`](../manifest.json).
- **Core** rules are always loaded by the UniKit AI toolkit. **Stack** rules are loaded on demand when the matching framework/module is detected.
